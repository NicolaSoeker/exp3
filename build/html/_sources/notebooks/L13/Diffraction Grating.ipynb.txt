    "We consider a number of $N$ slits of width $b$. The slits have a distance $d$ from each other. Each slit acts like the slit before giving rise to a diffraction pattern, that is oscillating with decreasing amplitude. The width of this diffraction pattern is determined by $\\lambda/b$ and the pattern is given by\n",
    "\n",
    "$$\n",
    "I(\\theta)=I_s\\frac{\\sin^2\\left (\\pi \\frac{b}{\\lambda}\\sin(\\theta)\\right)}{\\left( \\pi \\frac{b}{\\lambda}\\sin(\\theta)\\right)^2}\n",
    "$$"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "Now we have multiple of these \"slit\" sources. Each Huygens source in a slit has a corresponding Huygens source in a neighboring slit at a distance $d$. So these pairs of sources interfer like in our multiple wave interference \n",
    "\n",
    "$$\n",
    "I=I_{0}\\frac{\\sin^2(N\\phi/2)}{\\sin^2(\\phi/2)}\n",
    "$$\n",
    "\n",
    "where the phase difference between neighboring waves is given by \n",
    "\n",
    "$$\n",
    "    \\phi=k\\Delta s=\\frac{2\\pi}{\\lambda}d\\sin(\\theta)\n",
    "$$\n",
    "\n",
    "which finally gives\n",
    "\n",
    "$$\n",
    "I=I_{0}\\frac{\\sin^2(N\\pi\\frac{d}{\\lambda}\\sin(\\theta))}{\\sin^2(\\pi\\frac{d}{\\lambda}\\sin(\\theta))}\n",
    "$$"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "The intensity distribution behind a diffraction grating is now given as the product of the two contributions of single slit diffraction and multiple source interference."
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "\n",
    "<div class=\"alert alert-info\">\n",
    "\n",
    "**Diffraction Grating**\n",
    "\n",
    "The intensity distribution generated by a diffraction grating from monochromatic light and observed in the far field is given by\n",
    "    \n",
    "$$\n",
    "I=I_{0}\\frac{\\sin^2\\left (\\pi \\frac{b}{\\lambda}\\sin(\\theta)\\right)}{\\left( \\pi \\frac{b}{\\lambda}\\sin(\\theta)\\right)^2}\\frac{\\sin^2(N\\pi\\frac{d}{\\lambda}\\sin(\\theta))}{\\sin^2(\\pi\\frac{d}{\\lambda}\\sin(\\theta))}\n",
    "$$\n",
    "    \n",
    "where $\\lambda$ is the wavelength of the light and $b$ the width of the slit, $d$ the distance of the slits and $N$ the number of slits illuminated. The angle of observation is given by $\\theta$. \n",
    "    \n",
    "</div>"
